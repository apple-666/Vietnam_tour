# 胡志明市4天旅游助手 🇻🇳

这是一个纯前端开发的旅游助手应用，用于展示胡志明市4天3晚旅游计划的地图视图。

## 功能特点

- 📍 **地图展示**：在交互式地图上显示酒店和所有景点的位置
- 🎯 **景点信息**：点击标记查看详细的景点介绍、开放时间和亮点
- 📅 **行程管理**：按天查看行程安排，点击可定位到当天的景点
- 🛤️ **路线规划**：可视化显示每天的游览路线
- ⚠️ **重要提示**：展示旅游过程中的注意事项和实用信息

## 快速开始

### 方法1：直接在浏览器中打开（推荐）

1. 双击 `index.html` 文件，或
2. 右键 `index.html` → 选择浏览器打开

### 方法2：使用本地服务器（推荐用于开发）

如果你安装了 Python：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器访问：`http://localhost:8000`

### 方法3：使用 VS Code Live Server

1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键 `index.html` → "Open with Live Server"

## 数据说明

所有旅游数据存储在 `data.js` 文件中，包括：

- **hotels**: 3个可选酒店的详细信息
- **attractions**: 10个主要景点的信息和坐标
- **itinerary**: 4天详细的行程安排
- **tips**: 旅游注意事项和实用信息
- **emergencyContacts**: 紧急联系方式

## 自定义和扩展

### 修改旅游数据

编辑 `data.js` 文件可以自定义：
- 酒店信息
- 景点位置和描述
- 行程安排
- 提示信息

### 调整样式

编辑 `style.css` 文件可以自定义：
- 颜色主题
- 布局和间距
- 响应式设计断点

### 添加新功能

在 `app.js` 中可以扩展功能，例如：
- 添加更多交互功能
- 集成其他地图服务
- 添加数据导出功能

## 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式和响应式设计
- **JavaScript (ES6+)**: 应用逻辑
- **Leaflet.js**: 开源地图库

## 浏览器兼容性

支持所有现代浏览器：
- Chrome/Edge (推荐)
- Firefox
- Safari

## 注意事项

- 本应用使用 OpenStreetMap 的地图数据，需要互联网连接
- 所有坐标数据均基于公开信息，实际位置可能有细微偏差
- 建议在出行前再次确认景点的开放时间
